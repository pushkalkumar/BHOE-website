# A Short History of Nearly Everything — AP Chemistry Final Project

An interactive website connecting Bill Bryson's *A Short History of Nearly Everything* to AP Chemistry curriculum concepts.

## Pages
- **Home** — overview and key stats
- **The Big Bang** — matter formation, plasma state, nucleosynthesis
- **Stars as Reactors** — stellar fusion, binding energy, element synthesis
- **Atomic Models** — interactive timeline from Dalton (1803) to quantum (1926)
- **Radioactive Decay** — alpha/beta/gamma decay + half-life calculator
- **Matter at Scale** — intermolecular forces, phase changes
- **AP Chem Hub** — curriculum connections to CED units and Big Ideas
- **Citations** — MLA-formatted sources

## Tech
Plain HTML, CSS, and JavaScript — no frameworks.

## Source
Bryson, Bill. *A Short History of Nearly Everything.* Broadway Books, 2003.
